# First Principles

Strip every assumption, analogy, and convention until only fundamental truths remain. Rebuild from there.

## Method
1. Identify the claim or approach
2. Ask "Why?" repeatedly until you hit bedrock (physics, math, human nature)
3. Discard everything that is "because that is how it is done"
4. Rebuild from the bedrock truths only
5. Compare rebuilt version to original. Differences = convention, not necessity.

"Best practice" is not a first principle. It is someone else's conclusion.
